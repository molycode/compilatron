# Application Icons

Pre-generated from the source artwork (`tools/ctrn.png`).

| File | Size | Used for |
|------|------|----------|
| `app_icon.png` | 32×32 | Linux hicolor icon theme (32×32 slot) |
| `app_icon_48.png` | 48×48 | Linux hicolor icon theme (48×48 slot) |
| `app_icon_128.png` | 128×128 | Linux hicolor icon theme (128×128 slot) |
| `app_icon_256.png` | 256×256 | Linux hicolor icon theme (256×256 slot) |
| `app_icon.ico` | 16/32/48px | Windows executable resource (`app.rc`) |

To regenerate, run `tools/generate_icons.py` (see `tools/` — gitignored, developer only).
